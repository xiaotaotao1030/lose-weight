# 轻盈计划

个人减脂管理小程序的网页版原型。

## 技术方案

- 纯 HTML / CSS / JavaScript
- 不需要安装依赖
- 适合先做原型和新手维护

## 目录结构

```text
.
├── index.html
├── manifest.json
├── sw.js
├── assets
│   └── app-icon.svg
├── src
│   ├── scripts
│   │   ├── body-checkin.js
│   │   ├── calculator.js
│   │   ├── daily-record.js
│   │   ├── food-ai.js
│   │   └── app.js
│   └── styles
│       └── base.css
```

## 打开方式

直接用浏览器打开 `index.html`。

## AI 功能说明

当前 AI 饮食识别是前端原型模拟结果，不会上传图片。

真实上线时需要增加后端接口，由后端调用视觉 AI 服务。API Key 不能写在前端代码里。

## 身体照片打卡说明

当前身体照片只保存在当前浏览器本地。清理浏览器缓存或换设备后，记录可能丢失。

## 每日记录说明

每日体重、饮食和运动记录保存在当前浏览器本地。切换设备或清理浏览器数据后，记录可能丢失。

## 添加到手机主屏幕

iPhone Safari 打开页面后，可以通过“分享”菜单选择“添加到主屏幕”。

当前直接打开本地 `index.html` 可以预览界面；如需完整 PWA 离线能力，需要通过本地服务器或 HTTPS 部署访问。
